# CodeAlpha_Credit_Scoring_Model
Credit scoring model using machine learning to predict customer creditworthiness based on financial data like income, debt, and payment history.
# 💳 Credit Scoring Model

## 📌 Objective

Predict an individual's creditworthiness using machine learning based on financial data.

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## 🧠 Approach

* Used Logistic Regression for classification
* Performed feature engineering (debt-to-income ratio)
* Split dataset into training and testing sets

## 📊 Features Used

* Income
* Debt
* Payment History
* Debt-to-Income Ratio (engineered feature)

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC

## 🚀 How It Works

1. User inputs financial details
2. Model processes input
3. Predicts whether customer is:

   * ✅ Creditworthy
   * ❌ Risky

## ▶️ How to Run

```bash
pip install pandas numpy scikit-learn
python credit_model.py
```

## 📌 Output Example

* Good Customer (Loan Approved)
* Risky Customer (Loan Rejected)
